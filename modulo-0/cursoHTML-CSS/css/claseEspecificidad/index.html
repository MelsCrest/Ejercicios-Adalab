Cuantos más selectores se usen, más especificidad se da. Ej: #page-title .title / #main-nav .nav / .nav li / .nav .blog.

No es recomendable usar tanto 'id' para dar estilo, es preferible llamar a otra clase para dar más especificidad. Ej: #main-nav .blog → .nav .blog.
