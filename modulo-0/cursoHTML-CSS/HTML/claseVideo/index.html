Etiqueta 'video' y 'source'

Etiqueta 'video'

Atributos 'controls' y 'preload'

  • El atributo 'controls' muestra los controles (play, stop...).
  • El atributo 'preload="auto"' hace que el vídeo se empiece a descargar en el momento que se carga la página

Etiqueta 'source'

Atributo 'src'

  • Creamos tantas etiquetas 'source' como formatos de vídeo tengamos. El navegador elegirá el formato que pueda reproducir.
  • Añadiendo a continuación del nombre de la ruta '#t=10,30' del src, elegimos el segundo que queremos que empiece el vídeo y el segundo en que pare.
