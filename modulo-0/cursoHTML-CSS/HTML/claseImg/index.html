Formatos de imagen

Formatos lossless

  • Formatos sin pérdida.
  • No pierde calidad la imagen.
  • Suelen ser pesadas.
  • GIF, PNG 8(logotipos, iconos fondo transparente, 256 colores), PNG 24(más de 256 colores, más pesado)

Formatos lossy

  • Menos calidad.
  • Se puede modificar el tamaño.
  • Menor tiempo de carga.
  • JPG/JPEG (para fotos, colores ilimitados, se puede comprimir aunque pierda calidad), SVG (iconos, logos. Se puede escalar sin perder calidad)

Preferible trabajar con Lossy

Optimización de imágenes

  • Tamaño promedio 70kb.
  • Para comprimir el tamaño de la imagen: tiny png (funciona con png y jpg).
  • Para retirar los metadatos de la imagen (de aquellas echas con el móvil o cámaras que añaden esos datos): verexif.
  • Galería de imágenes: pexels.

Uso de las etiquetas 'figure', 'img' y 'figcaption'
